# Portfolio

Personal portfolio site built with React, TanStack Start, Tailwind CSS, Shadcn UI, and Framer Motion.

## Live

[pranishc.com.np](https://pranishc.com.np)

## Stack

- **Framework** — React + Vite
- **Routing** — TanStack Router (with View Transition API for page animations)
- **Styling** — Tailwind CSS + Shadcn UI
- **Animation** — Framer Motion
- **Icons** — Lucide React + Simple Icons

## Features

- Page transitions using the View Transition API
- Expandable animated navigation tab bar
- Dark / light mode toggle
- Skills section, project showcase, work history, and blog

## Running locally

```bash
pnpm install
pnpm dev
```
